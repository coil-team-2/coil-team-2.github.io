Project Overview

Our team has created a movie information search. This consists of leveraging AI to provide movie rating predictions, and creating a microservice on Microsoft Azure to access this model, along with other information about the movies. We created a web page that allows users to navigate this information intuitively. The movie search microservice is hosted on a Azure cloud app services in order to use the CI/CD pipeline to ensure easy scaling and access. For a more in depth report on how this microservice was created, check out this report. The front-end UI and back-end cloud api to properly integrate the movie predictions will be developed by the University of Dayton group consisting of Tyler and Justin, while the data mining and movie prediction application will be developed by the Vietnam National University HCM City group consisting of John, Luke, and Jane.
